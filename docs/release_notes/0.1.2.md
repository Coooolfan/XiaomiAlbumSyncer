# v0.1.2

此版本允许您在下载截图等媒体文件时，向其中填充Exif数据，以方便第三方程序为这些文件生成时间线。

**默认不开启，请手动前往设置开启该功能。源文件本身存在Exif数据时，程序无论如何都不会向其填充**
